# Film Festival Game - ಪಡೆದುಕೊಳ್ಳಿ ಚಿತ್ರ ರಹಸ್ಯ

An interactive movie guessing game featuring films from Kannada, Bollywood, Tollywood, Hollywood, and Anime.

## Features
- Guess movies from blurred frames
- Multiple difficulty levels
- Score tracking
- Responsive design
- Multi-language support

## How to Play
1. Guess the movie from the blurred image
2. Use "Reveal More" to make the image clearer
3. Type your guess and submit
4. Earn points for correct answers

## Deployment
This game can be easily deployed on Vercel, Netlify, or any static hosting service.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Responsive Design
